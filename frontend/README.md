# OPC Frontend

这是 OPC（One-Person Company）项目的前端 Demo，目标是把“一人公司正在运转”的叙事落到可浏览、可跳转、可联动的页面体验里。

## 当前重点

- 总部大楼与核心办公室主链路
- Agent 档案展示
- A2A 协作可视化
- 旗舰项目推进链路

## 启动方式

```bash
npm install
npm run dev
```

默认访问地址为 [http://localhost:3000](http://localhost:3000)。

## 目录说明

- `app/`：页面路由与场景入口
- `components/`：业务组件与办公室界面模块
- `lib/`：共享数据、schema 与辅助逻辑
- `public/`：静态资源
